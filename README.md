# Multiplanetary Homestead Manifesto

## Preamble

The purpose of this manifesto is to establish a set of values & principles to guide the open-source development of appropriate technologies (including both the life-support systems and tools) needed to sustain small-scale intensive permaculture homesteads as to eliminate supplychain dependencies (and by extension, artificial scarcity) - enabling post-scarcity economies, while preserving biodiversity and regenerating ecosystems.

## Values of a Solarpunk Multiplanetary Homestead

1. regenerative (over non-regenerative) resource extraction<sup>[1](#myfootnote1)</sup>
2. circular (over linear) product lifecycle<sup>[1](#myfootnote1)</sup>
3. recursively (over shallow) open source development<sup>[2](#myfootnote2)</sup><sup>[3](#myfootnote3)</sup>
4. localized/distributed (over centralized) production<sup>[3](#myfootnote3)</sup>

### Permaculture Ethics

<a name="myfootnote1">1</a>: Earth-care

<a name="myfootnote2">2</a>: People-care

<a name="myfootnote3">3</a>: Fair-share
